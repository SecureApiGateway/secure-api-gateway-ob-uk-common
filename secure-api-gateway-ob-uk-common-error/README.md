# securebanking-openbanking-uk-common

A Java library containing error related code used by two or more Secure API Gateway UK repositories.

### How to Build

This is a Java Maven project, built using JDK 14.

#### Prerequisites

You need the following on your development machine;
Java 14 or later
Maven 3.6.0 or later

#### Building

```
git checkout git@github.com:SecureApiGateway/secure-api-gateway-ob-uk-common.git
cd secure-api-gateway-ob-uk-common/secure-api-gateway-ob-uk-common-error
mvn clean install
```
